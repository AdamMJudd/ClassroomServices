# Windows 2016 Base Image

## Hardware Support

Supported:

*   Dell T5400
*   Dell 990
*   Dell 9010
*   Dell 9020
*   Dell 7040
*   Lenovo 7484
*   Lenovo 8811
*   Lenovo 9088

Not Supported:

*   Dell GX270 (x86 Only)
*   Dell GX280
*   Dell GX620

## Requirements

*   DNS Resource Record
   *   Type: `TXT`
   *   Name: `sitename`
   *   Note: Just the site name on a single line.
*   Dell 7040 Requires a new WinPE wim file in WDS.
*   Run the Ghost Scan script to tag images as being supported.
